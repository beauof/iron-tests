Order of command line parameters:
----------------------------------------------
- geometry
1. width (x) - real
2. height (y) - real
3. length (z) - real

- mesh
4. number elements (x) - integer
5. number elements (y) - integer
6. number elements (z) - integer
7. interpolation order - integer

- solver
8. solver type - integer

- material
9. e modulus - real
10. poisson ratio - real

- bc
11. displacement (x) - real
